# DigiReps Marketplace

The premium standard for digital services. Vetted talent, professional delivery.

## Core Features
- **Elite Discovery**: Curated talent across high-impact digital sectors.
- **Secure Workflows**: On-platform communication and simulated escrow payments.
- **Quality Control**: Vetted top 1% freelancers only.

## Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Technology Stack
- **Framework**: React 18 with Vite
- **Styling**: Tailwind CSS & Shadcn/UI
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Backend**: Supabase
